Profiles of the builders, researchers, investors, and catalysts shaping the OpenClaw agentic AI ecosystem. From creator Peter Steinberger to NanoClaw's Gavriel Cohen, Andrej Karpathy's foundational influence, and the research teams advancing the frontier of autonomous agents.

// FILE_001 Creator

Peter Steinberger

Founder and Lead Developer·OpenClaw, OpenAI

Austrian engineer and creator of OpenClaw, one of the fastest-growing open-source AI agent projects. Previously built PSPDFKit. Now shaping agentic AI infrastructure and governance as the project scales.

// OPENCLAW_LINK
Creator, technical direction, governance
linkedin github x.com
// FILE_002 Catalyst

Andrej Karpathy

Founder·Eureka Labs

Slovakian-Canadian AI researcher born in Bratislava, who co-founded OpenAI, later served as Director of AI and Autopilot Vision at Tesla, and in 2024 founded Eureka Labs, an AI education platform. Holds a PhD from Stanford (advised by Fei-Fei Li), authored the widely used CS231n deep learning course, and coined the term "vibe coding" in February 2025 — the development philosophy that directly enabled OpenClaw's creation. Runs the "Zero to Hero" YouTube series on LLM fundamentals with millions of views.

// OPENCLAW_LINK
An active OpenClaw user who automates his home using a claw agent. His viral X post praising NanoClaw as having a codebase that "fits into both my head and that of AI agents" was the direct catalyst for NanoClaw's breakout growth — 22,000 stars and 4,600 forks in a matter of weeks.
linkedin website github x.com
// FILE_003 Maintainer

Vincent Koc

AI Research Engineer and Developer Relations·Comet

A globally recognized lecturer, futurist, and keynote speaker with over two decades of experience across finance, telecommunications, travel, and FMCG. Has led data-driven projects for Qantas, McDonald's, Cisco, and the Australian Federal Government. Teaches applied data science and generative AI at MIT, UT Austin, and UNSW.

// OPENCLAW_LINK
One of a small handful of official OpenClaw maintainers. Responsible for agents, telemetry, hooks, and security contributions. Contributed roughly 20% of the core security patches in a major OpenClaw release, and maintains the awesome-openclaw curated ecosystem resource list.
linkedin github x.com
// FILE_004 Investor

Dave Morin

Investor·Offline Ventures · Slow Ventures · Path

Founding board member of the OpenClaw Foundation, where he helps guide governance and ecosystem development as the project evolves from an open-source experiment into a more structured AI agent platform.

// OPENCLAW_LINK
Board member of the OpenClaw Foundation
linkedin github x.com
// FILE_005 Builder

Gavriel Cohen

Co-founder & Creator·NanoCo (formerly Qwibit)

BSc in Physics and Computer Science from Tel Aviv University, former developer team lead at Wix.com, and ex-VP at Concrete Media. Co-founded an AI-native go-to-market agency with his brother Lazer Cohen that was on track for $1M ARR before he shut it down to focus full-time on NanoClaw.

// OPENCLAW_LINK
Built NanoClaw in a weekend as a tiny, secure alternative to OpenClaw after discovering it had stored all his personal WhatsApp messages in unencrypted plaintext.
linkedin website github x.com
// FILE_006 Builder

Matt Schlicht

Founder·Moltbook

Serial entrepreneur and builder who moved to Silicon Valley in 2008 without a college degree. Previously co-founded Octane AI, an ecommerce AI agent platform focused on quiz commerce for Shopify brands. Forbes 30 Under 30 (twice listed). Known for rapid, vibe-coded shipping.

// OPENCLAW_LINK
Launched Moltbook on January 27, 2026 — a social networking service intended to be used by AI agents such as OpenClaw.
linkedin github x.com
// FILE_007 Org

Sipeed

Founders / Team·Shenzhen Sipeed Tech Ltd.

A Shenzhen-based RISC-V hardware company known for affordable development boards and AI cameras. Makes embedded hardware including the LicheeRV Nano ($10 RISC-V board) and MaixCAM. Well-established in China's open-source hardware community before OpenClaw existed.

// OPENCLAW_LINK
Built PicoClaw in a single day on February 9, 2026 — a Go rewrite of the OpenClaw concept designed to run on $10 hardware with under 10MB RAM.
website github x.com

// FILE_011 Builder

Mario Zechner

Independent software developer, coach & angel investor·Independent

Veteran open-source developer with 15+ years across academia, game engines (creator of libGDX), applied ML, compilers, and graphics. Author of books and papers, conference speaker, and award winner (Oracle JavaOne Rock Star, Duke's Choice). Built Pi (pi-mono) — a minimalist, opinionated coding-agent harness with just four core tools, a tiny system prompt, unified LLM API, and TUI — which became the actual runtime powering OpenClaw's agent loop and skill execution.

// OPENCLAW_LINK
Provided the core agent runtime (Pi) that enables OpenClaw's autonomous task execution; key technical collaborator and dependency. Specially thanked in the OpenClaw repo.
